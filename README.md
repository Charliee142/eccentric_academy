# eccentric_academy
This is my source code for Eccentric Academy (A Plan-Based Cybersecurity Learning Platform Built with Django)

Project Name
============
Eccentric Academy – Ethical Hacking & Cybersecurity Training Platform


Description
-----------
Eccentric Academy is a full-featured online cybersecurity academy built with Django.
It implements a modern plan-based enrollment system where users subscribe to a pricing
plan and gain full access to all courses under that plan.

The platform focuses on ethical hacking education, hands-on learning, secure payments,
and clean UI/UX inspired by real-world bootcamps.


Problem it Solves
-----------------
Many online learning platforms:
- Force users to buy courses one by one
- Lack proper access control after payment
- Have weak security and poor UX
- Do not scale well for academies

Eccentric Academy solves this by:
- Using plan-based access (Beginner, Intermediate, Advanced)
- Providing full access to all courses under a plan
- Enforcing secure authentication and payment verification
- Offering a clean, modern, conversion-focused UI


Features
--------
✔ Plan-Based Enrollment (Pay once, access all courses)
✔ Pricing Page with Course Listings per Plan
✔ Secure Payment Integration (Paystack)
✔ Django Allauth Authentication
✔ Access Control (Already Own This Plan)
✔ AJAX Contact Form with Auto Email Reply
✔ Spam Protection (Honeypot / reCAPTCHA)
✔ Admin Dashboard for Managing Plans & Courses
✔ Responsive Bootstrap UI
✔ Countdown & CTA Conversion Elements

Installation
------------
1. Clone the repository
   git clone https://github.com/Charliee142/eccentric_academy.git

2. Navigate into the project
   cd eccentric_academy

3. Create a virtual environment
   python -m venv venv
   source venv/bin/activate  (Linux/Mac)
   venv\Scripts\activate     (Windows)

4. Install dependencies
   pip install -r requirements.txt

5. Set environment variables
   SECRET_KEY=your_secret_key
   PAYSTACK_SECRET_KEY=your_key

6. Run migrations
   python manage.py migrate

7. Create superuser
   python manage.py createsuperuser

8. Run server
   python manage.py runserver

9. Visit
   http://127.0.0.1:8000
